# Pandas Practice Exercises

## Overview

This repository contains practice exercises focused on data analysis with Python and Pandas. The notebook includes question-based analysis tasks using multiple datasets such as football team statistics, vehicle survey data, and transaction data.

The main goal of this repository is to practice core Pandas operations and improve data analysis problem-solving skills.

## Datasets Used

The notebook includes exercises based on the following datasets:

- Euro 2012 Team Statistics
- Auto Survey Dataset
- Bread Basket Transaction Dataset

## Topics Covered

- Reading CSV files with Pandas
- Inspecting datasets
- Cleaning column names
- Filtering rows
- Sorting values
- Grouping data with `groupby`
- Creating pivot tables
- Using `query`
- Aggregating values with `agg`
- Creating new calculated columns
- Finding maximum and minimum values
- Basic transaction analysis


## Technologies Used

- Python
- Pandas
- NumPy
- Jupyter Notebook

## Project Status

Completed as a Pandas practice and data analysis exercise notebook.
